## ¿Para qué sirve?

El módulo de Inspecciones te permite realizar verificaciones periódicas de tu establecimiento usando formularios con preguntas predefinidas. Cada respuesta queda registrada y, si se detecta algún problema, el sistema genera una acción correctiva automáticamente.

## Configuración inicial

Antes de realizar inspecciones, crea los tipos de inspección en [**Configuración → Inspecciones**](/configuracion/inspecciones-config):

<img src="/images/configuracion/inspecciones.png" alt="Configuración de tipos de inspección" />

Para cada tipo de inspección puedes:
- Ponerle un nombre descriptivo
- Agregar preguntas de distintos tipos: texto, número, fecha o selección múltiple
- Marcar qué preguntas son obligatorias
- Ordenar las preguntas como prefieras

## Cómo realizar una inspección

1. Ve al módulo **Inspecciones** desde el menú lateral
2. Verás la lista de tipos de inspección disponibles

<img src="/images/inspecciones/tipos.png" alt="Lista de tipos de inspección disponibles" />

3. Haz clic en el tipo de inspección que quieres realizar
4. Responde cada pregunta del formulario

<img src="/images/inspecciones/formulario.png" alt="Formulario de inspección" />

5. Si detectas un problema en alguna pregunta, el sistema te pedirá que asignes una acción correctiva: elige quién debe resolverlo y establece una fecha límite
6. Al terminar, haz clic en **Enviar**

## Acciones correctivas

Cuando una respuesta indica un problema, el sistema crea automáticamente una Acción Correctiva. Puedes ver todas las acciones pendientes en el módulo [**Actividades**](/actividades), en la sección de Acciones Correctivas.

Cada acción correctiva incluye:
- Descripción del problema encontrado
- La acción que se debe tomar
- Quién es responsable de resolverla
- Fecha límite

## Cómo ver el historial

En la misma página de Inspecciones, puedes consultar todas las inspecciones realizadas anteriormente. Usa los filtros de tipo, fecha y usuario para encontrar inspecciones específicas.
