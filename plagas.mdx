## ¿Para qué sirve?

El módulo de Control de Plagas te permite registrar el estado de cada zona respecto a plagas: si se detectó alguna presencia, qué tipo, qué acción se tomó y cómo están los elementos de control (trampas, cebos, etc.). También puedes registrar las visitas del servicio de control de plagas externo.

## Configuración inicial

No se requiere configuración adicional para usar este módulo. Solo asegúrate de tener las **zonas** configuradas en [**Configuración → Establecimiento**](/configuracion/establecimiento), ya que el registro se hace por zona.

## Cómo registrar un control de plagas

1. Ve al módulo **Plagas** desde el menú lateral
2. Verás el formulario de registro en la pestaña **Registro**

<img src="/images/plagas/registro.png" alt="Formulario de registro de control de plagas" />

3. Completa los campos:
   - **Zona**: selecciona el área que estás inspeccionando
   - **¿Detectaste presencia de plagas?**: selecciona Sí o No
     - Si seleccionas **Sí**: indica el tipo de plaga (insectos, roedores, aves, hongos u otro), describe el hallazgo y la acción inmediata tomada
   - **Estado de los elementos de control** (trampas, cebos, etc.): indica si están en buen o mal estado
     - Si están en mal estado: describe el problema y qué acción se requiere
   - **Estado del área**: indica si está limpia y ordenada, o si hay acumulación de desorden
4. Haz clic en **Guardar**

## Acciones correctivas

Si registras presencia de plagas o elementos de control en mal estado, el sistema puede generar una Acción Correctiva automáticamente. Puedes verla y gestionarla desde el módulo [**Actividades**](/actividades).

## Cómo ver el historial

Haz clic en la pestaña **Historial** para consultar los registros anteriores de cada zona.

<img src="/images/plagas/historial.png" alt="Historial de control de plagas" />

Puedes filtrar por zona, tipo de plaga y rango de fechas para analizar patrones.

## Control externo de plagas

Si contratas una empresa externa de control de plagas, puedes registrar sus visitas en la pestaña **Control Externo**.

<img src="/images/plagas/externo.png" alt="Control externo de plagas" />

Aquí puedes:
- Registrar la empresa prestadora del servicio
- Indicar la fecha de la última visita
- Ver cuándo está programada la próxima visita
- Adjuntar certificados o informes del servicio
- Identificar visitas vencidas (en rojo) o próximas (en amarillo)
