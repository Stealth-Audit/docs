## ¿Para qué sirve?

Aquí diseñas los formularios de inspección que usará tu equipo. Cada tipo de inspección tiene sus propias preguntas, y puedes crear tantos tipos como necesites (ej: inspección diaria de higiene, inspección de infraestructura, inspección de BPM).

## Cómo acceder

Ve a **Configuración → Inspecciones** desde el menú lateral.

<img src="/images/configuracion/inspecciones.png" alt="Configuración de inspecciones" />

## Cómo crear un tipo de inspección

1. Haz clic en **Nuevo Tipo**
2. Se abrirá el formulario:

<img src="/images/configuracion/inspecciones-form.png" alt="Formulario para crear un tipo de inspección" />

3. Escribe el nombre y una descripción del tipo de inspección
4. Haz clic en **Guardar**

## Cómo agregar preguntas

Una vez creado el tipo de inspección, haz clic sobre él para ver sus preguntas y haz clic en **Agregar Pregunta**:

<img src="/images/configuracion/inspecciones-detalle.png" alt="Detalle de un tipo de inspección con sus preguntas" />

Se abrirá el formulario de pregunta:

<img src="/images/configuracion/inspecciones-pregunta-form.png" alt="Formulario para agregar una pregunta" />

Para cada pregunta:
- Escribe el texto de la pregunta
- Elige el tipo de respuesta:
  - **Texto**: el operador escribe una respuesta libre
  - **Número**: el operador ingresa un valor numérico
  - **Fecha**: el operador selecciona una fecha
  - **Selección múltiple**: el operador elige entre opciones predefinidas
- Marca si la pregunta es **obligatoria** o no
- Haz clic en **Guardar**

Repite este proceso para cada pregunta que necesites. Puedes reordenarlas arrastrándolas.

## Cómo editar o desactivar una inspección

Haz clic en el botón **Editar** junto al tipo de inspección que quieres modificar. Puedes cambiar el nombre y la descripción en cualquier momento. Si ya no quieres que aparezca disponible, desactívala desde el mismo formulario.

<Note>
Los registros históricos de una inspección desactivada se conservan. Solo deja de aparecer disponible para nuevas inspecciones.
</Note>
