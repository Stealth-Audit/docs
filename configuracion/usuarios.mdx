## ¿Para qué sirve?

Desde esta sección puedes crear, editar y desactivar los usuarios que tienen acceso a tu plataforma. También puedes organizar a los usuarios en grupos para asignarles tareas de forma colectiva.

## Cómo acceder

Ve a **Configuración → Usuarios** desde el menú lateral.

<img src="/images/configuracion/usuarios.png" alt="Gestión de usuarios" />

## Cómo crear un usuario

1. Haz clic en el botón **Nuevo Usuario**
2. Se abrirá el formulario de creación:

<img src="/images/configuracion/usuarios-form.png" alt="Formulario para crear un usuario" />

3. Completa la información:
   - **Nombre completo**
   - **Correo electrónico** (se usará para iniciar sesión)
   - **Número de empleado** (opcional)
   - **Teléfono** (opcional)
   - **Rol**: elige entre Administrador, Supervisor u Operador
4. Haz clic en **Guardar**

El nuevo usuario recibirá un correo para establecer su contraseña.

## Roles de usuario

| Rol | Accesos |
|-----|---------|
| **Administrador** | Acceso completo: puede configurar la plataforma, ver todo y gestionar usuarios |
| **Supervisor** | Puede ver todos los registros y gestionar actividades, pero no puede cambiar la configuración |
| **Operador** | Solo puede registrar información del día a día |

## Cómo editar o desactivar un usuario

Haz clic en el usuario que quieres modificar. Puedes actualizar su información o cambiar su estado a **Inactivo** si ya no hace parte del equipo.

## Grupos de usuarios

En la pestaña **Grupos** puedes crear conjuntos de usuarios para asignarles tareas colectivamente. Por ejemplo, puedes crear un grupo "Turno Mañana" y asignarle actividades específicas a ese grupo.

<img src="/images/configuracion/grupos.png" alt="Gestión de grupos de usuarios" />

Para crear un grupo:
1. Haz clic en **Nuevo Grupo**
2. Se abrirá el formulario de creación:

<img src="/images/configuracion/grupos-form.png" alt="Formulario para crear un grupo" />

3. Ponle un nombre al grupo
4. Agrega los usuarios que pertenecen a él
5. Haz clic en **Guardar**
